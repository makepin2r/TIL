## HTTP 메서드 설계의 기준
- 가장 중요한 것은 **리소스 식별**
### 리소스
- 데이터 개념 자체
  - 회원(==리소스) 등록/수정/조회(행위)
- 리소스를 식별하는 좋은 방식
  - 리소스 자체만 식별 → 리소스만 URI에 매핑
  - 등록, 수정, 조회 등의 동작은 모두 배제
